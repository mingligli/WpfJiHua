项目介绍
DesktopPlanWidget 是一款轻量级桌面计划管理挂件，专为需要高效管理日常日程、清晰查看待办事项的用户设计。程序以悬浮窗口形式呈现，支持鼠标穿透、桌面置底，不遮挡桌面图标和其他应用，同时具备计划管理、外观自定义、开机自启、数据备份恢复等实用功能，帮你轻松掌控每日日程，提升时间管理效率。
核心定位：轻量、便捷、不打扰，让计划管理融入桌面，无需切换应用即可查看和操作待办事项。
核心功能
- 计划展示与管理：清晰展示未来指定天数的待办计划，支持计划新增、编辑、删除，勾选完成状态，自动排序显示。
- 日期格式化展示：采用「公历+农历+星期」三合一显示方式，直观呈现计划日期，兼顾实用性与传统习惯。
- 外观自定义：可自由设置标题、日期、内容的文字颜色，调节文本透明度，适配不同桌面风格。
- 窗口特殊功能：默认桌面置底、鼠标穿透，按住Ctrl键可取消穿透并操作窗口；支持窗口固定在屏幕右上角，不占用核心操作区域。
- 开机自启：可设置开机自动启动，无需手动打开，启动后自动加载之前的配置和计划。
- 数据备份与恢复：支持将计划数据和应用配置备份为.bak文件，可随时恢复，防止数据丢失。
- 重复计划自动刷新：每日凌晨1点自动刷新重复计划，无需手动操作，确保计划连续性。
安装与运行
环境要求
 - 操作系统：Windows 10 及以上版本（32位/64位均可）
 - .NET Framework 4.8 及以上（若运行失败，需先安装对应框架）
安装步骤
1. 下载程序压缩包，解压至任意文件夹（建议路径无中文、无空格，如：D:\DesktopPlanWidget）。
2. 找到解压文件夹中的 DesktopPlanWidget.exe 文件，双击即可运行程序。
3. （可选）运行后在设置面板勾选「开机启动」，下次开机将自动启动程序。
首次运行说明
首次运行时，程序将自动生成默认配置文件和计划数据文件，窗口默认显示在屏幕右上角，默认显示未来7天的计划，文本透明度为1.0（不透明），颜色为默认白色（标题、内容）和黄色（日期）。
使用说明
基础操作
- 窗口操作：默认鼠标穿透，无法点击；按住Ctrl键可取消穿透，此时可拖动窗口、点击按钮。
- 查看计划：窗口默认显示未来7天的待办计划，按日期排序，每条计划包含「日期（公历+农历+星期）」和「待办内容」，勾选复选框可标记计划为完成状态。
- 打开设置面板：点击窗口右上角的「⚙」按钮，即可打开设置面板，进行各项功能配置。
详细功能操作
1. 计划管理
在设置面板中点击「计划管理」，将打开计划编辑窗口，可进行以下操作：
- 新增计划：输入计划内容、选择计划日期，点击保存即可添加新计划。
- 编辑计划：选中已有计划，修改内容或日期后保存。
- 删除计划：选中不需要的计划，点击删除按钮即可移除。
- 重复计划：支持设置重复计划（如每日、每周重复），程序将每日凌晨自动刷新下一次重复计划。
2. 显示天数设置
在设置面板中点击「设置天数」，弹出输入窗口，输入1-30之间的整数，点击确定，即可修改窗口显示的未来计划天数（默认7天）。设置后将立即生效，并自动保存配置。
3. 外观设置
在设置面板中点击「外观设置」，将打开外观配置窗口，可分别设置：
- 标题颜色：设置窗口顶部「📅 日程」的文字颜色（支持十六进制颜色码）。
- 日期颜色：设置计划中「日期」部分的文字颜色。
- 内容颜色：设置计划中「待办内容」和复选框的文字颜色。
设置完成后点击确定，外观将立即刷新，并保存配置。若颜色解析失败，将自动使用默认颜色（标题/内容为白色，日期为黄色）。
4. 文本透明度调节
在设置面板中拖动「文字透明度」滑块，可调节整个窗口的文本透明度（范围0.2-1.0），拖动时实时预览效果，松开后自动保存配置。透明度越低，窗口越不遮挡桌面内容。
5. 开机自启设置
在设置面板中勾选「开机启动」，即可开启开机自启功能；取消勾选则关闭。开启后，下次开机时程序将自动启动并加载之前的配置和计划。
6. 数据备份与恢复
- 备份数据：在设置面板中点击「备份数据」，选择备份文件保存路径，程序将自动备份当前所有计划和配置，生成以「backup_日期」命名的.bak文件。
- 恢复数据：在设置面板中点击「恢复数据」，选择已有的.bak备份文件，程序将恢复备份中的计划和配置，恢复后立即生效。
7. 进入官网
在设置面板中点击「进入官网」，将自动打开系统默认浏览器，跳转到官方网站（可在后台代码中修改官网地址）。
8. 退出程序
在设置面板中点击「退出」，即可关闭程序。程序关闭时将自动保存当前配置和计划，下次启动时恢复。
配置说明
程序配置文件由系统自动生成和管理，无需手动修改，路径为程序所在目录下的「Config.json」，主要包含以下配置项：
- ShowDays：计划显示天数（1-30）
- TextOpacity：文本透明度（0.2-1.0）
- TitleColor：标题颜色（十六进制颜色码）
- DateColor：日期颜色（十六进制颜色码）
- ContentColor：内容颜色（十六进制颜色码）
- AutoStart：开机自启状态（true/false）
若配置文件损坏，程序将自动生成默认配置，不影响正常使用。
常见问题（FAQ）
1. 问题：程序运行后，看不到窗口或无法点击？
解决方法：程序默认鼠标穿透，按住Ctrl键即可取消穿透，此时可看到窗口并进行操作；若仍看不到，检查是否被其他窗口遮挡，或重启程序。
2. 问题：开机自启设置后，下次开机未自动启动？
解决方法：检查系统权限，确保程序所在文件夹有读写权限；若系统开启了开机启动管理，将程序添加到允许开机启动的列表中。
3. 问题：备份/恢复数据失败？
解决方法：检查备份文件路径是否正确，确保备份文件未损坏；恢复时确保程序处于正常运行状态，且备份文件是本程序生成的.bak文件。
4. 问题：点击「进入官网」无法打开网页？
解决方法：检查网络连接，确保浏览器可正常访问网络；若仍无法打开，可在后台代码中修改官网地址为正确的URL。
5. 问题：颜色设置后不生效？
解决方法：确保输入的颜色码为正确的十六进制格式（如#FFFFFF），若格式错误，程序将自动使用默认颜色；设置后点击确定，即可立即生效。
注意事项
- 程序运行时，建议不要删除程序所在目录下的配置文件（Config.json）和数据文件，否则会导致配置丢失、计划消失。
- 若需要卸载程序，直接删除程序所在文件夹即可，无残留文件（开机自启需先取消勾选，否则注册表中会残留自启项）。
- 修改程序后台代码后，需重新编译生成.exe文件才能生效，建议修改前备份原始代码。
- 程序仅支持Windows系统，暂不支持macOS、Linux等其他操作系统。
